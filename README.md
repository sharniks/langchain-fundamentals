# 🚀 LangChain Learning Repository

A hands-on repository containing my practice, experiments, and mini-projects while learning **LangChain** and **Generative AI**.

This repository follows the **CampusX LangChain Tutorial** and includes additional experiments, debugging notes, and improvements that I implemented while learning.

---

## 📌 About

The objective of this repository is to build a strong understanding of LangChain fundamentals before moving on to production-ready AI applications such as RAG systems, AI Agents, and LangGraph workflows.

Topics covered include:

* LLM Integration
* Chat Models
* Prompt Engineering
* Prompt Templates
* ChatPromptTemplate
* Message Placeholder
* Output Parsers
* Structured Output
* Pydantic Output Parser
* Hugging Face Integration
* Google Gemini Integration
* Streamlit Applications
* Text Splitters (RecursiveCharacterTextSplitter)
* Embeddings (HuggingFace, BGE, MPNet, MiniLM)
* Vector Stores (ChromaDB, FAISS)
* Similarity Search & Similarity Search with Score
* Retrieval-Augmented Generation (RAG) fundamentals
* YouTube Transcript Extraction for RAG pipelines
* Tool / Function Calling with LLMs
* LangChain Agents (ReAct)

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/<your-username>/<repository-name>.git
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate it

**Windows**

```bash
.venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```text
OPENAI_API_KEY=your_api_key
GOOGLE_API_KEY=your_api_key
HF_TOKEN=your_token
ANTHROPIC_API_KEY=your_api_key
```
